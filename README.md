## Web Hosting and Domain Name Concepts

### 1. Domain Name
This is a unique address that represents your identity or brand on the internet It translates complex IP addresses into a human-readable format that visitors can easily type into their browser to find your website.

### 2. Web Hosting
This is a service that provides the storage and technology needed to make your website's files accessible on the internet .A hosting provider stores your HTML, CSS, and media files on a server so that your website can be viewed online 24/7.
